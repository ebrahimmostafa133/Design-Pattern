Prototype Pattern
=================

Purpose
-------
Specify the kinds of objects to create using a prototypical instance, and create new objects by copying
this prototype. Useful when object creation is expensive or complex.

This folder contains a small Java example:

- `Prototype.java` — prototype interface
- `ConcretePrototype.java` — implements `clone()`
- `PrototypeDemo.java` — demonstrates cloning

How to compile & run
---------------------
```
cd "Desktop/ITI/Design Pattern Study/creational/prototype/java"
javac *.java
java PrototypeDemo
```
